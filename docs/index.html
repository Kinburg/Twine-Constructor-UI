Що таке Purl?

What is Purl?

Purl — це десктопний застосунок для Windows, macOS та Linux, який дозволяє створювати інтерактивні текстові ігри та візуальні новели у форматі SugarCube 2 без написання коду. Ви будуєте історію з візуальних блоків, а Purl генерує готовий .twee файл для Tweego або Twine, або .html файл та папку із файлами.

Purl is a desktop app for Windows, macOS and Linux that lets you create interactive fiction and visual novels in SugarCube 2 format without writing code. Build your story from visual blocks, and Purl generates a ready-to-use .twee file for Tweego or Twine, or .html file and files folder.

Purl активно розвивається — нові блоки та функції регулярно з'являються у кожному оновленні.

Purl is actively developed — new blocks and features are regularly added with every update.

Можливості Features

Все що потрібно для інтерактивної прози

Everything you need for interactive fiction

Блоковий редактор

Block editor

Додавайте та впорядковуйте блоки перетягуванням. Кожна сцена — окремий список блоків.

Add and reorder blocks by drag & drop. Each scene is a separate list of blocks.

Діалоги та тексти

Dialogue & text

Блоки діалогів з персонажами, кольорами та аватарами. Ефекти друкарської машинки та затримки.

Dialogue blocks with characters, colors and avatars. Typewriter and delay effects.

Вибори та умови

Choices & conditions

Блоки вибору, кнопки, посилання та умовні гілки з підтримкою вкладених блоків.

Choice blocks, buttons, links and conditional branches with nested block support.

Змінні та логіка

Variables & logic

Числа, рядки, булеві, масиви. Вирази, маппінг, режими random та dynamic.

Numbers, strings, booleans, arrays. Expressions, mapping, random and dynamic modes.

Медіа

Media

Блоки зображень, відео та аудіо. Статичний або динамічний режим для зображень; фонова музика та звуки з контролем гучності для всієї історії.

Image, video and audio blocks. Static or bound mode for images; background music and sounds with a master volume control for the whole story.

Експорт у .twee

Export to .twee

Генерація готового .twee файлу для Tweego або імпорту в Twine 2.

Generates a ready .twee file for Tweego or import into Twine 2.

Експорт у HTML + assets

Export to HTML + assets

Генерує самодостатній HTML-файл разом із папкою assets. Готово до публікації — відкривається у браузері без жодних додаткових інструментів.

Generates a standalone HTML file together with an assets folder. Ready to publish — opens in any browser with no extra tools needed.

Спостерігачі (Watchers)

Watchers

Фонове відстеження змінних: задайте умову та дії, які виконуються автоматично при зміні сцени або натисканні кнопки.

Background variable tracking: set a condition and actions that fire automatically on scene change or button click.

Повзунок гучності

Volume slider

Елемент бокової панелі — повзунок гучності з опціональною кнопкою вимкнення звуку. Синхронізує гучність усього аудіо та відео в історії.

Sidebar panel widget — a volume slider with an optional mute button. Syncs the volume of all audio tracks and video elements in the story.

Undo / Redo

Повна історія дій до 100 кроків. Ctrl+Z / Ctrl+Shift+Z або кнопки в панелі інструментів.

Full action history up to 100 steps. Ctrl+Z / Ctrl+Shift+Z or toolbar buttons.

Пошук по проєкту

Project search

Пошук по тексту сцен, блоків та по використанню змінних. Перехід до результату одним кліком.

Search by scene text, blocks and variable usage. Click to navigate to any result.

Типи блоків Block types

20 типів блоків

20 block types

Кожен блок — готовий елемент сцени. Комбінуйте їх для будь-якого жанру.

Each block is a ready-to-use scene element. Combine them for any genre.

Текст Text

Абзац тексту з підтримкою розмітки SugarCube. Live-режим для автоматичного оновлення при зміні змінних. Є ефект друкарської машинки.

Text paragraph with SugarCube markup support. Live mode for auto-refresh when variables change. Typewriter effect available.

Діалог Dialogue

Репліка персонажа з аватаром, ім'ям та кольоровим оформленням. Підтримує вирівнювання, вкладені блоки та ефект друкарської машинки.

Character speech bubble with avatar, name and color styling. Supports alignment, nested blocks and typewriter effect.

Вибір Choice

Меню варіантів для переходу між сценами. Кожен варіант може мати умову відображення.

Branching menu that lets the player pick a destination scene. Each option can have a visibility condition.

Умова (IF) Condition (IF)

Гілки if / else-if / else з підтримкою вкладених блоків будь-якого типу. Підтримує числові діапазони та операції з масивами.

if / else-if / else branches with nested block support. Handles numeric range checks and array operations.

Встановити змінну Set variable

Задає значення змінній: вручну, випадково, через вираз або за значенням іншої змінної. Підтримує операції з масивами (push, remove, clear).

Assigns a value to a variable: manually, randomly, via expression, or by mapping another variable. Supports array operations (push, remove, clear).

Зображення Image

Статичне зображення за URL або шляхом до ресурсу. У режимі прив'язки зображення змінюється залежно від значення змінної.

Static image by URL or asset path. In bound mode the image switches based on a variable's value.

Відео Video

Вбудоване відео з налаштуваннями автовідтворення, повтору та відображення елементів керування.

Embedded video with configurable autoplay, loop and player controls.

Аудіо Audio

Фонова музика або звуковий ефект. Миттєве або відкладене відтворення, цикл, гучність, зупинка при виході зі сцени або продовження як глобальний трек.

Background music or sound effect. Immediate or delayed playback, loop, volume, stop-on-leave or persist as a global track. Can silence all other sounds before playing.

Кнопка Button

Стилізована кнопка, що змінює змінні без переходу між сценами. Може оновити поточну сцену після натискання.

Styled button that mutates variables without navigating. Can trigger a full scene refresh after the click.

Посилання Link

Стилізована кнопка для переходу в іншу сцену або повернення назад. Може змінювати змінні перед переходом.

Styled navigation button that goes to another scene or back. Can mutate variables before navigating.

Поле вводу Input field

Текстове або числове поле, що зберігає введене гравцем значення у змінну. Підтримує запис в елемент масиву.

Text or number input that saves the player's entry to a variable. Supports writing to an array element by index.

Прапорці Checkbox

Група прапорців у двох режимах: кожен прапорець керує окремою булевою змінною, або всі разом — одним масивом.

Checkbox group in two modes: each checkbox controls its own boolean variable, or all together manage a single array.

Перемикачі Radio

Група перемикачів, що задає рядкову змінну значенням обраного варіанту. Рівно один варіант завжди обраний.

Radio button group that sets a string variable to the selected option's value. Exactly one option is always chosen.

Таблиця Table

Сітка клітинок з текстом, змінними, прогрес-барами, зображеннями та кнопками. Ідеальна для панелей статистики чи інвентарю.

Grid of cells containing text, variables, progress bars, images and buttons. Ideal for stats panels or inventory displays.

Роздільник Divider

Горизонтальна лінія з налаштуванням кольору, товщини та вертикальних відступів.

Horizontal rule with configurable color, thickness and vertical margin.

Включення сцени Include

Вставляє вміст іншої сцени через <<include>>. Підтримує обгортку з налаштуванням розміру, рамки та фону.

Inserts another scene's content via <<include>>. Supports a styled wrapper with size, border and background options.

Виклик функції Function

Виконує сцену-функцію без переходу — змінні оновлюються, але гравець залишається на місці. Зручно для повторюваної логіки.

Executes a function-tagged scene without navigating — variables update, player stays in place. Great for reusable logic.

Спливаюче вікно Popup

Відкриває модальне вікно SugarCube Dialog при рендері пасажу. Вміст береться зі сцени з тегом «popup». Кнопки також можуть відкривати попапи.

Opens a SugarCube Dialog modal when the passage renders. Content comes from a scene tagged "popup". Buttons can open popups too.

Raw Code

Вставляє довільні макроси SugarCube або HTML безпосередньо в експортований пасаж, без жодних перетворень.

Inserts arbitrary SugarCube macros or HTML verbatim into the exported passage, without any transformation.

Нотатка Note

Коментар розробника, видимий лише в редакторі, не потрапляє в експорт. Зручний для позначок та пояснень логіки.

Developer comment visible only in the editor, never exported. Useful for inline annotations and logic notes.

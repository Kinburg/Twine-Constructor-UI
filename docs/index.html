Common questions

Часті запитання

Is Purl really free? No hidden fees? Purl справді безкоштовний? Без прихованих платежів?

Yes, completely. No subscription, no premium tier, no paywalled blocks — every feature is available to everyone. If Purl has been useful, you can support development with a voluntary donation, but it is entirely optional.

Так, повністю. Без підписки, преміуму чи платних блоків — всі можливості доступні всім. Якщо Purl виявився корисним, можна підтримати розробника донатом, але це повністю на ваш розсуд.

Do I need to know how to code? Чи потрібно вміти програмувати?

No. Purl is a visual, block-based editor — you build scenes by combining ready-made blocks without writing any code. If you do know SugarCube or HTML, you can also drop in a Raw Code block anywhere for full control.

Ні. Purl — візуальний блоковий редактор: сцени збираються з готових блоків без жодного коду. Якщо ви знаєте SugarCube або HTML — блок «Сирий код» дасть повний контроль у будь-якому місці.

What is SugarCube / Twine? Що таке SugarCube / Twine?

Twine is an open-source tool for creating interactive fiction. SugarCube is a story format that runs Twine games in any browser. Purl exports your project as a ready-to-play SugarCube 2 file — no extra setup needed for players.

Twine — це інструмент з відкритим кодом для створення інтерактивної фантастики. SugarCube — формат, у якому Twine-ігри запускаються у браузері. Purl експортує проєкт як готовий SugarCube 2-файл — гравцям не потрібно нічого додатково встановлювати.

Does it work on Mac and Linux? Чи працює на Mac та Linux?

Yes. Purl is a desktop application available for Windows, macOS and Linux. Download the build for your platform from the Download section.

Так. Purl — десктопний застосунок для Windows, macOS і Linux. Завантажте збірку для своєї платформи в розділі Download.

Where are my project files stored? Де зберігаються мої проєкти?

Entirely on your machine. Projects are saved as .purlproj files wherever you choose. Nothing is sent to the cloud — Purl has no backend and no accounts.

Повністю на вашому комп'ютері. Проєкти зберігаються як файли .purlproj у вибраному вами місці. Нічого не відправляється в хмару — у Purl немає бекенду і акаунтів.

Can I use Purl for commercial projects? Чи можна використовувати Purl у комерційних проєктах?

Yes. Purl is released under the MIT license, which allows commercial use, modification and redistribution. Everything you create with it belongs to you.

Так. Purl поширюється за MIT-ліцензією, яка дозволяє комерційне використання, модифікацію та розповсюдження. Весь контент, який ви створюєте, належить вам.
